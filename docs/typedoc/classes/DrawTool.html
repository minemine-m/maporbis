    Class DrawTool

    DrawTool：通用绘图工具

    负责：

    • 处理 Map 的 DOM 事件（click/mousemove/dblclick）
    • 维护顶点序列 _clickCoords
    • 调用模式定义的 create/update/generate
    • 向外触发：drawstart / drawvertex / drawend / drawing 等事件
